Course Requirements and Evaluation:

● Readings
First, find a Github repository in this topic. Then read the paper or blog with the algorithm implemented in the repository. Finally, summarize the mathematics behind the algorithm and describe the data used. ● Homework
Carry out the assigned tutorial, one/two will be assigned in each class. The homework is due once a week at the beginning of each lecture.The homeworks are listed under resources. Turn in your assignment by submitting to the class GitHub.
